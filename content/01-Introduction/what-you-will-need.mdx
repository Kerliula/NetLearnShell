# What do we need to get connected?

This course runs entirely in the browser. No software to install, no virtual
machines, no package managers.

## A modern browser

Chrome, Firefox, or Safari — any of them will do. The interactive terminals and
diagrams run directly in the page.

## A curious mind

Seriously, that is it. This course assumes no prior networking knowledge. It
does assume you are willing to slow down and think when something is not
immediately obvious — networking rewards patience.

## Helpful but not required

These things will make some sections feel more familiar, but you will not be
lost without them:

- Basic command line experience — knowing what a terminal is and how to type a command
- Some programming background — understanding that code runs on machines that talk to other machines
- Having seen an IP address before, even if you do not know what it means

## What about the textbook?

*Computer Networks: A Top-Down Approach* by Kurose & Ross is the backbone of
this course. You do not need a copy to follow along — everything here stands on
its own. But if at any point you want to go deeper, the book is the natural next
step.
