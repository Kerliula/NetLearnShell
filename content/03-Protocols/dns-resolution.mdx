## DNS Resolution

**DNS** (Domain Name System) translates human-readable domain names into IP addresses.

### Resolution steps

1. Browser checks local cache
2. Query sent to recursive resolver
3. Root server directs to TLD server
4. TLD server directs to authoritative nameserver
5. Nameserver returns the IP address

> DNS is often called the "phonebook of the internet".
